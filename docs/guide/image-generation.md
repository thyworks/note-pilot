# アイキャッチ画像を生成する

記事を作成すると、記事と同時に **1280×670px** のアイキャッチ画像が自動生成されます。**Gemini 3.1 Flash Image**（`gemini-3.1-flash-image-preview`）を使用します。

`GEMINI_API_KEY` が未設定の場合は画像生成をスキップして記事作成・校閲・投稿を通常通り続行します。

## 事前設定

### 1. Gemini API キーを取得する

API キーの取得手順は [Gemini API キーの取得](./gemini-api-key) をご覧ください。取得したキーを次のステップで設定します。

### 2. `.env` ファイルにキーを書き込む

`note-pilot/skills/gemini-image/.env` をテキストエディタで開き、API キーを記入します。

```
GEMINI_API_KEY=AIzaSy...（ここにキーを貼り付ける）
```

::: warning この設定はファイルを直接編集してください
API キーはセキュリティ上の理由から対話形式では設定できません。
テキストエディタで `.env` ファイルを直接編集してください。
:::

## スタイルテンプレートで画像を生成する

記事作成時、アイキャッチ画像のスタイルを 4 種類から選べます。

| No. | スタイル名 | 特徴 |
|-----|-----------|------|
| 1 | ミニマルフラット | 白背景・ボーダー枠・線画人物 |
| 2 | 教育的インフォグラフィック | グラデ背景・フラットアイコン・キャラクター |
| 3 | ダークマーケティング | 黒背景・アニメキャラ・ゴールドエフェクト |
| 4 | リアルビジネス | オフィス写真・プレゼン風 |

各スタイルのサンプル画像は `note-pilot/templete-image/` フォルダに保存されています。

### 選択フロー

記事作成時、Claude が以下の手順でスタイルを確認します。

**Step 1 — スタイルを選ぶ**

前回選んだスタイルが「（前回）」付きで表示されます。初回はデフォルト（スタイル 1）が選択されます。番号を答えるだけで OK です。

```
アイキャッチのスタイルを選んでください。
[1] ミニマルフラット（白背景・ボーダー枠・線画人物）　← （デフォルト）
    note-pilot/templete-image/templete2_...webp
[2] 教育的インフォグラフィック（グラデ背景・フラットアイコン・キャラクター）
    note-pilot/templete-image/templete1_...webp
[3] ダークマーケティング（黒背景・アニメキャラ・ゴールドエフェクト）
    note-pilot/templete-image/templete3_...png
[4] リアルビジネス（オフィス写真・プレゼン風）
    note-pilot/templete-image/templete4_...webp
```

**Step 2 — タイトルと色味を確認（1往復で完結）**

- 「タイトルテキストを画像に入れますか？」
- 「色味や雰囲気で変えたい点はありますか？」

この 2 点にまとめて答えると、すぐに生成が始まります。

### 前回選択の記憶

一度スタイルを選ぶと、アカウント設定ファイルの `eyecatch.preferred_template` に番号が保存されます。次回以降は「（前回）」として表示されるため、毎回の選択が楽になります。

## 生成された画像の保存先

記事ファイルと同じ `output_dir` の `images/` フォルダに保存されます。

```
{output_dir}/
├── thy-works-article-041-note-paste.md
└── images/
    └── thy-works-article-041-eyecatch.png   ← ここに保存
```

## 投稿時の自動アップロード

保存された画像は、自動投稿の際に note のアイキャッチとして**自動でアップロード**されます。手動でのアップロード操作は不要です。

```
記事作成 → 画像生成（images/ に保存）→ 校閲 → 投稿（アイキャッチも自動アップロード）
```

画像が存在しない場合（API キー未設定などでスキップされた場合）は、アイキャッチなしで投稿処理が続行されます。

## 依存パッケージについて

初回実行時に `note-pilot/skills/gemini-image/` で自動的に `npm install` が実行されます。Node.js 18 以上が必要です。

手動でインストールする場合:

```bash
cd note-pilot/skills/gemini-image
npm install
```

## トラブルシューティング

| 症状 | 対処 |
|------|------|
| 「APIキー未設定」でスキップされる | `skills/gemini-image/.env` に `GEMINI_API_KEY` を記入したか確認 |
| 401 / 403 エラー | API キーが無効。[Google AI Studio](https://aistudio.google.com/app/apikey) で再発行 |
| 429 エラー（クォータ超過）| 利用上限に達した。Google AI Studio の課金状況を確認してください |
| タイムアウトで失敗する | 自動で最大 10 回リトライします。繰り返し失敗する場合はしばらく待ってから再試行してください |
| `sharp が見つかりません` | `cd note-pilot/skills/gemini-image && npm install` を実行 |
