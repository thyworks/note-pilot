# ナレッジを追加する — `/np:learn`

記事を公開したあとや、使っていて気づいたことをその都度 note-pilot に伝えましょう。蓄積されたナレッジは次の記事作成・タグ選択・テーマ選びに活用されます。

## 公開済み記事の URL を登録する

記事を note.com で公開したら、URL を note-pilot に教えます。

```
記事を公開しました。URL は https://note.com/your-id/n/nxxxxxxxx です
```

登録されると：

- アカウント設定の**公開済みリスト**に追加される
- 次回から**関連記事**として自動でリンクされる
- 記事の連番が自動で更新される

## 学びを記録する

タグの反響・記事の反応・書き方の改善点など、気づいたことを自由に伝えます。

```
「生成AI」タグは「AI活用」より反応が薄かった
```

```
導入部に具体例を入れると読まれやすい気がする
```

```
副業の話は絶対に混入しないよう徹底して
```

伝えた内容はアカウント設定のメモセクションに記録され、次の記事作成時に参照されます。

## 呼び出し方

```
/np:learn
```

```
ナレッジを追加して
```

```
〇〇を覚えておいて
```

```
記事URLを登録して
```

## 登録内容の確認

アカウント設定ファイル（`accounts/{slug}.md`）をテキストエディタで開くと、公開済みリストと蓄積されたメモを確認できます。
