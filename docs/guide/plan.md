# 連載計画を立てる

書くテーマが思いつかないときや、次の記事の方向性を決めたいときに使います。

## 基本的な使い方

```
次のテーマを考えて
```

```
連載を計画して
```

```
ネタを出して
```

過去に書いた記事と重複しない新テーマを **3〜5案** 提案します。気に入った案を選ぶと `next_topics` の先頭に追加され、次回の記事作成時に自動で選ばれます。

## 提案の質を上げるには

公開済み記事を登録しておくと（[ナレッジを追加する](./knowledge)）、すでに書いたテーマを避けた提案になります。また、アカウントのターゲット読者・テーマ設定が詳しいほど、より的を絞った提案になります。

## 呼び出し方

```
次のテーマを考えて
```

```
ネタが尽きてきた、何を書けばいい？
```

## テーマを手動で追加する

提案以外のテーマを追加したい場合は、以下のように伝えます。

```
次のテーマに「Gemini APIの使い方」を追加して
```

アカウント設定ファイル（`accounts/{slug}.md`）の `next_topics` に直接書き込むこともできます。
