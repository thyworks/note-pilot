features:
  - icon: 👤
    title: アカウント管理
    details: noteアカウントなしでも記事を書き溜められます。preparing → active の移行で、今まで書いた記事がそのまま投稿可能に。複数アカウントの切り替えも対応。
  - icon: ✍️
    title: 記事作成
    details: アカウント設定とナレッジベースを参照して、ルール準拠の記事を自動生成。比喩・CTAも含めて一発で完成します。
  - icon: 🔍
    title: 校閲チェック
    details: 30項目以上のチェックリストを自動検査。問題があれば即修正、全項目OKになるまで繰り返します。
  - icon: 🚀
    title: 自動投稿
    details: Playwrightで実際のChromeを操作。タイトル・本文・タグ・目次・埋め込みカードを自動入力して下書き保存。
  - icon: 📚
    title: ナレッジ追加
    details: 公開済み記事URLや「効果があったタグ」などをアカウント設定に蓄積。次の記事作成に活かします。
  - icon: 📅
    title: 連載計画
    details: 過去記事と重複しないトピック案を3〜5個提案。選んだテーマは自動でnext_topicsに追加されます。
